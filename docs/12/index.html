1.데이터 세부 조회
데이터를 수정(Update)하는 기능을 추가하려고 한다.
먼저 데이터 조회 화면에서 Uri에 id값이 추가되면 해당 id의 데이터만 조회하도록 한다.

2.데이터 수정 1
데이터를 수정하기에 앞서 DB에서 값을 받아와야한다.
그렇기에 edit 메소드에서 DB의 값을 받아오고 form에 각각의 콜롬값을 뿌려준다.

3.데이터 수정 2
form의 값을 수정하고 수정버튼을 누르면
DB에 쿼리가 실행되고 값이 수정된다.

4.데이터 삭제 기능 추가 1
수정 기능에서 삭제를 할 수 있는 기능을 추가해보자.

5.데이터 삭제 기능 추가 2
쿼리 변수를 만들어서 DELETE 기능을 하도록 만들고 실행한다.

6.리스트 수정
부트스트랩을 적용하고 한 페이지에 표시되는 콜롬값의 최대갯수를 제한한다.
