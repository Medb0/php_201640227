1.Databases.php
Tables.php 처럼 DB 목록을 출력하는 파일을 만든다.

2.database.html
Table.php와 마찬가지로 Databases.php를 만들었으니
Resource에 html도 똑같이 만들어주고 경로를 수정한다.

3.Uri 체크
현재 Uri를 검사하여 접속된 Uri가 table로 접속하게된다.
Uri가 없으면 데이터가 없음으로 나온다.

4.테이블 접속
접속시에 Uri를 확인하여 해당 table을 출력한다.

5.Uri 체크 최적화
Uri를 체크를 하는 코드를 따로 빼서 파일로 만들어주고
어디서든지 사용할 수 있도록 클래스와 함수를 만들어둔다.

6.$_POST
PHP에서는 post로 넘어온 값을 $_POST로 사용할 수 있다.

7.데이터 삽입 기능 추가
insert.html을 만들고 폼을 만들어준다.
만든 폼에 값을 넣고 버튼을 누르면 값을 넣고 데이터 조회 페이지로 돌아온다.

8.삽입 기능 최적화
DB에서 받아온 값이 $row 변수에 들어가게 되는데
$row에 DB의 열을 객체로 넣어준다. 그 객체안의 Field값이 id 일 경우에
id는 자동으로 만들어지기 때문에 입력폼이 필요가 없으므로 continue로 다음으로 넘어가주는 코드를 추가한다.

9.폼(form) 생성 최적화
나중에 폼의 갯수를 일일히 추가하는 일이 없도록 for문에서 Field의 갯수 만큼 알아서 추가하도록 설정한다.

10. 삽입(INSERT) 쿼리 최적화
마찬가지로 쿼리도 for문안에서 한꺼번에 처리하도록 변경한다.
