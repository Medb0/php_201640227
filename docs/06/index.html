1.쿠키
쿠키는 접속한 사용자 컴퓨터에 저장되는 데이터이다.
접속자 브라우저가 실행되고 있는 경우에만 존재한다.
PHP에서는 간단한 쿠키 내부 함수를 통해 쿠키 값을 설정할 수 있다.

2.세션
세션은 쿠키의 문제점을 보완하기 위해 민감한 정보를 접속자의 컴퓨터가 아닌 서버에 저장하는 기술이다.
세션은 접속자의 브라우저가 실행되고 있는 동안 유지된다.

3. setcookie
쿠키를 설정할 수 있는 setcookie라는 내장함수가 있다.
setcookie의 파라매터는 쿠키가 저장될 변수명 , 저장될 값 , 시간 , 경로 로 이루어져있다.

4.scandir
scandir 함수는 해당 폴더에 있는 파일,폴더를 보여준다.
ex) array scandir ( string $directory [, int $sorting_order = SCANDIR_SORT_ASCENDING [, resource$context ]] )
두번째 인자는 출력될 정렬방식이다.
scandir() 함수 사용할 경우 print_r() 함수를 활용하여 확인이 가능하다.

5.is_dir
is_dir 함수는 입력된 값이 디렉토리인지 확인하는 역할을 한다.
ex)
if(is_dir($test_file)){
echo "디렉토리 임";
}

6.json_encode
PHP Array 또는 String을 거의 모든 프로그래밍 언어에서 호환되는 JSON String으로 묶어 반환하는 PHP 함수
첫 번째 명령어 $value는 JSON으로 묶을 Array 또는 String 값이다. String일 경우 곁따옴표가 양쪽에 붙는다.
두 번째 명령어 $options는 이스케이프할 문자를 설정할 수 있다.
